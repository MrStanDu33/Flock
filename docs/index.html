About

Flock is a personal project built using vanilla JavaScript to discover Flocks algorithms.

The goal is to learn how the flocks algorithm works to simulate a flock of birds (boids).

Usage

Installation

  • Clone the repository

      git clone https://github.com/MrStanDu33/Flock.git

  • Install dependencies

      npm install

  • Run Project

    • Build You can build the project by running

        npm run build

      You will then obtain a dist/ folder containing the built project, you now can open the index.html file in your browser.

    • Serve You can run the project in a development server

        npm run dev

      You will be able to access the game at http://localhost:8080

Lint

You can lint the project's code by running

  npm run lint
